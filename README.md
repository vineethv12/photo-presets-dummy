# pictureprefect

Indian wedding photography portfolio and Lightroom preset shop.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- Framer Motion
- Wouter (routing)
- Vanilla CSS (no Tailwind needed)
